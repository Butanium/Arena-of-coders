loop totalEntities
read entId:int entType:word(10) health:int shield:int action:word(20) targets:word(256) distEn:int borderDist:int borderDistRank:int distEnRank:int healthRank:int shieldRank:int totalRank:int



loop allyBotAlive
loop totalEntities
read entId:int entType:word(256) distMe:int distMeRank:int shieldComp:int healthComp:int totComp:int
write 0 MOVE 2;1 IDLE ;2 MOVE 8;3 ATTACK 9;4 FLEE 10;5 MOVE 10,11,12;

STATEMENT
Control your bots in order to destroy the enemy team !

INPUT
botPerPlayer: the amount of bot you control
mapSize: the map size in meters (just here in case the creator change the map size during the contest)
allyBotAlive: the amount of your bot which are still alive
totalEntities : the amount of entities in the arena
entId: the unique gameEntity id, stay the same for the whole game
entId: the unique gameEntity id
entType: the gameEntity type in a string. It can be ALLY | ENEMY
entType: the gameEntity type in a string. It can be SELF | ALLY | ENEMY
health: the approximate gameEntity health. Can be 0 | 25 | 50 | 75 | 100, 25 meaning that your life is >= 25% and < 50% of your max life
shield : the approximate gameEntity shield. Can be 0 | 1 | 25 | 50 | 75 | 100, 1 meaning that your shield is >= 1% and < 25% of your max shield and 0 that you have no more shield left
action: action executed by the gameEntity last turn
targets: list of the targets id targeted by the robot last turn ("id1;id2;id3...") if the gameEntity is a robot, else -1 (the target for IDLE is the robot itself)
distEn : the RANGE from to the closest enemy (if the entity is an enemy it returns
the range to its closest ally)
borderDist : approximate distance between the gameEntity and the closest border
distEnRank :  NOT USED IN THIS LEAGUE (it'll be a RANK so an int between 0 and entityCount)
borderDistRank :  NOT USED IN THIS LEAGUE (a RANK)
shieldRank:  NOT USED IN THIS LEAGUE (a RANK)
healthRank:  NOT USED IN THIS LEAGUE (a RANK)
totalRank:  NOT USED IN THIS LEAGUE (a RANK)
distMe : approximate distance between the target and the current bot. Can be 0 to 3 for short, medium, long and out of range
distMeRank : entities are sorted by ascending order based on their distance to the current bot
shieldComp: -1 if the gameEntity has more shield than the current bot, 0 if it's equal, 1 if your bot as more shield
healthComp: same as shieldComp but for the health
totComp: same as shieldComp but based on the sum of health+shield
OUTPUT
yourBotId ATTACK | MOVE | FLEE targetId
